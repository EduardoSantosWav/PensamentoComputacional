# Pensamento Computacional 🧠

Repositório dedicado ao armazenamento de atividades, laboratórios e desafios desenvolvidos na disciplina de **Pensamento Computacional** do curso de Análise e Desenvolvimento de Sistemas.

## 🎯 Objetivo

O foco desta disciplina é o desenvolvimento do raciocínio lógico e a capacidade de decompor problemas complexos em soluções algorítmicas, utilizando os quatro pilares fundamentais:

1.  **Decomposição:** Fragmentação de problemas em partes menores.
2.  **Reconhecimento de Padrões:** Identificação de tendências e similaridades.
3.  **Abstração:** Foco no que é essencial, ignorando detalhes desnecessários.
4.  **Algoritmos:** Criação de passos sequenciais para a resolução de problemas.
